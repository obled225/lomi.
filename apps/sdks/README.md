# lomi. SDKs

Official SDKs for the [lomi.](https://lomi.africa) payments API, automatically generated from the OpenAPI specification.

## Available SDKs

| Language/FW    |    Package   |    Installation    |   Directory   |
|----------------|--------------|--------------------|---------------|
| **TypeScript** | `@lomi./sdk` | `npm i @lomi./sdk` | [`ts/`](./ts) |
| **JavaScript** | `@lomi./sdk-js` | `npm i @lomi./sdk-js` | [`js/`](./js) |
| **Python**     | `lomi/sdk` | `pip install lomi/sdk` | [`python/`](./python) |
| **Go**         | `github.com/lomiafrica/lomi./lomi-go-sdk` | `go get github.com/lomiafrica/lomi./lomi-go-sdk` | [`go/`](./go) |
| **PHP**        | `lomi/sdk` | `composer require lomi/sdk` | [`php/`](./php) |
| **Laravel**    | `lomi/sdk-laravel` | `composer require lomi/sdk-laravel` | [`laravel/`](./laravel) |
| **Nextjs**     | `@lomi./sdk-next` | `npm i @lomi./sdk-next` | [`nextjs/`](./nextjs) |
